Jake v2.0


To package this into a Joomla extension:

- Copy the contents of this repo into the a temporary directory somewhere.
- Move administrator/components/com_jake/jake.xml to "." 
- Move administrator/components/com_jake to admin/
- Move components/com_jake to site/
- In Windows, multi-select jake.xml, admin/, and site/
- Right-click "Send to Compressed (zipped) folder" to zip it up and rename the resulting zipfile as desired
- In the Joomla Extension Manager, browse to the zipfile to install it.

For full configuration details, go to: http://blog.echothis.com/2012/09/26/jake-2-0-released/
