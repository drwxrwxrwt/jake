README
------

To package this into a Joomla extension:

- Copy this contents of this commit into the a temporary directory somewhere.
- Rename administrator/ to admin/
- Move admin/components/com_jake/com_jake-joomla_1.5.xmml to a sibling of admin/
- In Windows, multi-select admin/ and site/
- Click "Send to Compressed (zipped) folder...."
- Rename resulting zipfile as desired
- In the Joomla Plugin Manager, browse to the zipfile to install it.
